# Next AI
An AI-powered PDF helper

## Getting Started
1. Clone this repository
    ```shell
    git clone https://github.com/stevenysy/nextai.git
    ```
2. Enter the cloned directory
    ```shell
    cd nextai
    ```
3. Install the dependencies
   ```shell
    npm install    
    ```
4. Run the app
    ```shell
    npm run dev
    ```
